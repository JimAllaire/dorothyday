FAQ

What is a markdown (MD) file?
Markdown is a lightweight markup language use to read and write plain text then convert it into other formats, like HTML. Markdwon can easily be converted into pdf, docx, epub, html and other file formats.
What is a creative commons license?
Under a creative commons license you are free to use and remix the documents on this website. You agree to make derivative works available under the same terms. Please read the summary at Creative Commons Non-Commercial Share Alike 3.0 license.
What is the DDLW #?
DDLW stands for Dorothy Day Library on the Web, # stands for number, the digits were assigned during the digitization process. The DDLW #, if known, can be used to go to a document directly on the Search page. The DDLW # should also be included in citing a document's source.(See below)
How do I send corrections to documents?
Corrections may be submitted to catholicworker.org@gmail.com. Include the DDLW # of the document, the first few words of the paragraph containing the error, and a description of the error.
How should I cite the source of a document?
Sample citation for article #456:
Day, Dorothy. "What Dream Did They Dream? Utopia or Suffering?". The Catholic Worker, July-August 1947, 1, 4, 6, 8. Reprinted from "Dorothy Day Library on the Web" at URL: http://dorothyday.catholicworker.org/articles/456.html.
Why are four file formats included?
The four formats--PDF, HTML, DOCZ, and MD--are in common use and are versatile, easily editable and combined in derivative publications.
Who can I contact with questions?
Questions may be submitted to catholicworker.org@gmail.com.
How can I use the HTML output?
The HTML output is already in a format for easy inclusion in websites and blogs.
